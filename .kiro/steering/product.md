# Product Steering

## プロジェクトの目的

Slack MCP Server は、MCP (Model Context Protocol) プロトコルに準拠したサーバーとして、AI エージェントが Slack ワークスペース内のメッセージを効率的に検索・取得できるようにします。

## コア価値

- **MCP プロトコル準拠**: 標準的な MCP サーバー実装により、Claude Desktop やその他の MCP クライアントから利用可能
- **Slack メッセージ検索**: ワークスペース内のメッセージを検索し、AI エージェントがコンテキストを取得できる
- **レート制限対応**: Slack API のレート制限を適切に処理し、安定した動作を保証
- **統一エラーハンドリング**: 一貫したエラー処理戦略により、デバッグと運用を容易に

## 主要機能

### `search_messages` ツール

Slack ワークスペース内のメッセージを検索する唯一の MCP ツール。

- **クエリベース検索**: テキストクエリでメッセージを検索
- **チャンネルフィルタリング**: 特定のチャンネルに絞り込んだ検索（オプション）
- **ページネーション**: 大量の結果を適切に処理
- **メタデータ付与**: チャンネル名、ユーザー名などのコンテキスト情報を含む

## 設計原則

- **シンプルさ**: 単一のツール（`search_messages`）に焦点を当て、機能を明確に
- **信頼性**: エラーハンドリングとログ記録により、運用時の問題を早期に発見
- **拡張性**: サービス指向アーキテクチャにより、将来的な機能追加に対応可能

## 利用シナリオ

- AI エージェントが Slack の過去の会話を参照してコンテキストを取得
- 特定のトピックに関する議論を検索
- チャンネル単位での情報検索
